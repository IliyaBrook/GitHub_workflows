# Quick Notes

A simple note-taking web app built with vanilla HTML, SCSS, and JavaScript.

## Features

- Add notes with text input or Enter key
- Delete notes with × button
- Note counter display

## Files

- `index.html` — main page
- `style.scss` — styles source (SCSS)
- `style.css` — compiled CSS
- `app.js` — application logic

## Usage

Open `index.html` in any browser. No build tools or dependencies required.
