# GitHub Workflows Cheat Sheet

A collection of ready-to-use GitHub Actions workflows for quick copy-paste into projects.

## Contents

| Workflow | Description |
|----------|-------------|
| `stale-issues.yml` | Auto-close inactive issues and PRs |
| `telegram-notify.yml` | Send Telegram notifications on repo events |

## Usage

Copy the desired file from `.github/workflows/` into your project and adjust the parameters to fit your needs.
